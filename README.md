### 功能特点

- 可选择适合你的购买方案
- 可设置执行时长
- 可定时执行
   - 注意：需设置屏幕常亮
- 可设置异常响铃 - `防止出现 BUG，导致错过抢购时间`
- 可自定义购买方案 - `目前暂时支持 JSON 修改`
   - 流程有问题，自己也可以尝试处理了
